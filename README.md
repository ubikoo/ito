## ito

Tiny libraries for graphics and physics research:

 - **ito::str** : string manipulation.
 - **ito::file** : file IO.
 - **ito::math** : random number generator, vector and matrix algebra, etc.
 - **ito::cl** : interface to OpenCL.
 - **ito::gl** interface to OpenGL.
